### HOW TO RUN ###

There are two ways to run the code:

1. Run the following commands in the terminal:

    javac *.java
    java Main


2. If 'make' is installed on your system, you can run the following command in the terminal:

    make
    
    This will compile and run the code automatically.


### PARAMETERS ###

There are six noteable parameters at the top of the Main.java file. These are:
	
    Type NAME (default_value)
    -------------------------
    Integer RUNS (3);
    Boolean RUN_ILS (true);
    Boolean RUN_TABU (true);

    Integer ITERATION_MULTIPLIER (10);
    Integer TABU_LIST_SIZE = (10);
    Integer NEIGHBOURHOOD_SIZE = (5);
