### HOW TO RUN ###

There are two ways to run the code:

1. Double click to run the run.bat file in the main directory.

2. Open in terminal in the "Code" directory and run the following commands:

    javac *.java
    java Main

### PARAMETERS ###

There are six noteable parameters at the top of the Main.java file. These are:
	
    Type NAME (default_value)
    -------------------------
    Integer RUNS_COUNT (1);  # How many times each algorithm is run for each file.
    Boolean runGA (true);    # Whether or not to run the Genetic Algorithm.
    Boolean runASO (true);   # Whether or not to run the Ant System Optimisation.
